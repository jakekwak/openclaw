# Generated Docs Artifacts

이 baseline 산출물들은 저장소가 소유한 OpenClaw config schema와 번들 channel/plugin metadata에서 생성됩니다.

- `config-baseline.json`은 손으로 수정하지 마세요.
- `config-baseline.jsonl`은 손으로 수정하지 마세요.
- `plugin-sdk-api-baseline.json`은 손으로 수정하지 마세요.
- `plugin-sdk-api-baseline.jsonl`은 손으로 수정하지 마세요.
- config baseline 산출물 재생성: `pnpm config:docs:gen`
- config baseline 산출물 검증: `pnpm config:docs:check`
- Plugin SDK API baseline 산출물 재생성: `pnpm plugin-sdk:api:gen`
- Plugin SDK API baseline 산출물 검증: `pnpm plugin-sdk:api:check`
